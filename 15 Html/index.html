Understanding HTML 5

The term HTML5 is essentially a buzzword that refers to a set of modern web technologies. This includes the HTML Living Standard (new features, upgrades + other things/features not part of HTML), along with JavaScript APIs to enhance storage, multimedia, and hardware access.

HTML Standard [Living Standard > its update by time] > is a document that tells to the browser how HTML should work.
